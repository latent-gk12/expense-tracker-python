# Expense Tracker

A console-based Expense Tracker built using Python.

## Features

- Add Expense
- View Expenses
- Save Expenses to CSV
- Load Expenses from CSV
- Menu Driven Interface

## Concepts Used

- OOP
- File Handling
- CSV Module
- Lists
- Loops
- Conditional Statements
